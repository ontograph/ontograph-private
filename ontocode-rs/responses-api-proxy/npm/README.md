# @openai/ontocode-responses-api-proxy

<p align="center"><code>npm i -g @openai/ontocode-responses-api-proxy</code> to install <code>ontocode-responses-api-proxy</code></p>

This package distributes the prebuilt [Codex Responses API proxy binary](https://github.com/openai/codex/tree/main/ontocode-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/ontocode-responses-api-proxy.js --help
```

Refer to [`ontocode-rs/responses-api-proxy/README.md`](https://github.com/openai/codex/blob/main/ontocode-rs/responses-api-proxy/README.md) for detailed documentation.
